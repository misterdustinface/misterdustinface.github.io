-------------------------------------------------------------
LEFT PLAYER CONTROLS:
W key makes paddle go UP
S key makes paddle go DOWN

D key makes paddle GROW, but also SLOWS paddle down
A key restores paddle SIZE and SPEED
-------------------------------------------------------------
RIGHT PLAYER CONTROLS:
UP_ARROW key 	makes paddle go UP
DOWN_ARROW key 	makes paddle go DOWN

RIGHT_ARROW key makes paddle GROW, but also SLOWS paddle down
LEFT_ARROW key 	restores paddle SIZE and SPEED
-------------------------------------------------------------

SPACEBAR key ADDS more balls (up to 5) to the game.

-------------------------------------------------------------

FIRST PLAYER TO GET 10 POINTS WINS. THE GAME WILL THEN STOP.
